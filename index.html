Versu

Motivation

Once I went to a shopping center to buy some new clothing. But I was alone and I couldn't decide which clothing to buy because I wasn't sure which fits better to me. In that situation I would appreciate some hints from my friends. Then I discussed with friends and found out that many people experience the same situation every day. It's not always related to a context of fashion but also for example when you want to get some hints regarding a tourist destination for your holidays. Beside that, various agencies doing market research or public opinion survey about politicians would benefit from an app where they can put products and people would vote which they like more, etc. Therefore, I decided to implement a social network, which allows its users to publish votings in form of photos such that followers can vote on those votings to help a user to decide in a particular situation.

Overview
  • Developed an instagram-like app for Android that allows users to post votings in the form of photos.
  • Integrated Firebase Cloud Messaging to enable push notifications and Crashlytics to receive crash reports.
  • Developed RESTful dockerized backend for the app by using Java and Spring Boot framework.
  • Used Amazon S3 bucket to store the photos.
  • Competed in Startup Weekend Bratislava 2015 with the idea.
Utilized

Java, Spring Boot (MVC, Security, Data), JUnit, Hibernate, MySQL, Docker, Git, Android, Maven

Alarm 4u

Motivation

Everyone has experienced a situation when an alarm is ringing in the morning, but is dismissed by a not fully-awaken person. That causes that people are late to work, school, etc. In order to solve that problem I decided to develop an application, which will force users to be awake before being able to dismiss an alarm. Therefore, the users have to solve a simple equation to dismiss an alarm. But it might happen that user solves an equation and then fall asleep again. To avoid that scenario the application has a feature, which rechecks if user is awake one minute after s/he has dismissed the alarm.

Overview
  • Developed an alarm application for Android.
  • Published on Google Play since 09/2013 till 05/2018 with ratings over 4 stars
  • More than 1500 active installations in 2014 and 2015.
  • Targeted on devices with older versions of Android (2.3)
Utilized

Android, Gradle, Git

Save Flappy Bird

Motivation

Before implementing Save Flappy Bird I already had some experience in Java and Android and I was curious about game development. Also it wasn't clear to me how does cross-platform frameworks work. Therefore, I googled for a cross-platform game development frameworks and found this tutorial by Kilobolt. The tutorial explains how to implement Flappy bird game on Android, desktop, iOS and Web from scratch. I used the code from the tutorial as a codebase for the Save Flappy Bird game.

Overview
  • Developed a game in Java for Android, desktop and web by utilizing libGDX - a cross-platform game development framework.
  • Utilized the ZombieBird's source code as my codebase to avoid having to write the same code again.

Save Flappy Bird is a modified version of the famous Flappy Bird game. Your task in this game is to move the pipes such that the bird can fly between them. The bird bounces the bottom pipe to jump and fly further.

There are other birds in the game. If you collide with a larger bird it will eat part of your body so you will become smaller. You die as soon as you are too small. If you collide with a smaller bird then you eat it and you will become bigger. But if you are bigger then you are heavy and slow so the flying becomes more challenging. As the time flows the games speed increases. To slow the game speed down a bit you can eat a larger bird.

Utilized

Java, Gradle, Git, libGDX
